# Skill: Skill-Controladores

**Archivo**: `data/skills/Skill-Controladores.md`

## Proposito

Plantilla para documentos tecnicos de proyectos con microcontroladores y sistemas embebidos.

## Estructura que define

1. **Titulo** - Nombre del proyecto
2. **Planteamiento del problema** - Problematica que aborda
3. **Importancia del proyecto** - Justificacion
4. **Solucion al problema** - Propuesta concreta
5. **Stack tecnologico** - Microcontroladores, sensores, lenguajes
6. **Diagrama de arquitectura** - Placeholder para diagrama
7. **Conclusiones** - Resumen en maximo 44 palabras

## Incluye

- Tabla de contenido (TOC) con campo dinamico de Word
- Formato APA 7 sobre el cual se aplica esta estructura

## Uso tipico

Proyectos de:
- IoT (Internet de las Cosas)
- Automatizacion
- Robotica
- Sistemas de monitoreo con ESP32, Arduino, sensores
