DOCXIX
generador de documentos .docx con IA

Sistema de generación de documentos académicos basado en skills (plantillas) y archivos existentes, con inteligencia artificial para estructurar y redactar el contenido.

Permite crear, editar y gestionar skills, subir archivos de referencia y generar documentos .docx con formato APA 7.

El uso del sistema es responsabilidad del usuario. No se almacenan API keys en el servidor.

Para usar la IA necesitas configurar al menos una API key de Gemini. Ve al panel lateral derecho > acciones > API keys para agregarlas.
